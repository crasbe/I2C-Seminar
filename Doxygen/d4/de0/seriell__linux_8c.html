Linux-Interface für serielle Schnittstelle.

Diese Datei enthält die Funktionen, die zur Ansteuerung des seriellen Interfaces des USB-ITS-Gerätes notwendig sind.

int oeffne_port ( int  fd,
int  port 
)

Öffnet den seriellen Port.

Für das serielle Device wird die Termios-Struktur von Linux verwendet.

Parameter
fdFiledeskriptor des seriellen Devices
portPortnummer des zu öffnenden Ports
Rückgabe
neuer Filedeskriptor

Folgende Flags werden gesetzt:

  • C_IFLAG
    1. IXON, IXOFF, IXANY (=0): Software Flusskontrolle ausschalten
    2. IGNBRK (=0): Break-Condition ignorieren
  • C_CFLAG Einstellungen für 8N1
    1. PARENB (=0): Paritykontrolle ausschalten
    2. CSTOPB (=0): Ein Stopbit
    3. CSIZE (=CS8): auf Charaktergröße CS8 setzen
    4. CS8: 8 bit pro Zeichen
  • C_LFLAG
    1. ICANNON (=0): Mit Rohdaten arbeiten
    2. ECHO (=0): Empfangene Zeichen nicht wieder zurückgeben
    3. ISIG (=0): Steuersignale (z.B. SIGQUIT) ignorieren
  • C_OFLAG
    1. OPOST (=0): Mit Rohdaten arbeiten
  • C_CC: Control Characters
    1. VMIN (=0): Es müssen keine Zeichen gesendet werden
    2. VTIME (=10): Timeout auf eine Sekunde setzen
